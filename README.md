
# 插件简介

用于处理 markdown. 已实现功能如下:
- 生成 TOC.
- 标题自动编号.
- 快速插入剪切板中的图片, 并将图片存储于本地.
- 块元素前空一行.
- 使用英文标点替换中文标点.

# 设置

可以在 "设置/扩展/MarkdownHandler" 下进行设置.


# 常见问题

- 如果你已经设置了自动保存, 然后又勾选了 "Markdown Handler: Auto Handler When Save", 则会出现当前文档被频繁处理的问题.
